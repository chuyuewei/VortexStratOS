VortexStrat 入门指南

欢迎使用 VortexStrat 操作系统！本指南将帮助您快速上手系统的基本功能。

# 安装 VortexStrat
sudo ./install.sh --mode=standard

# 启动核心服务
vortex-service --start-all

# 配置网络
vortex-net --configure

查看完整文档请访问我们的文档中心。

VortexStrat API 文档

VortexStrat 提供了丰富的 API 接口，方便开发者构建应用程序。

// 创建一个 Vortex 任务
VortexTask task = new VortexTask();
task.setPriority(VortexPriority.HIGH);
task.setHandler(() => {
    // 任务处理逻辑
});
task.schedule();
